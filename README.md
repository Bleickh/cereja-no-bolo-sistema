# Sistema Cereja no Bolo

## Status do Projeto
Em desenvolvimento

## Tecnologias utilizadas
- Java
- Swing (JFrame)
- MySQL
- JDBC

## Desenvolvedor
- José Felipe

## Objetivo do software
Sistema desktop para gerenciamento de produtos e movimentações de uma confeitaria, permitindo controle de estoque e organização dos itens vendidos.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Registro de movimentação (entrada e saída)
- Interface gráfica com navegação entre telas
- Integração com banco de dados MySQL
